# videoWater

由于常常在网上搜集视频，然后将视频进行一系列的处理，然后发布到头条等自媒体，
视频处理往往要花大量的时间，现在写了个程序，可以将视频进行批量处理。

下面的任意一个操作，都会修改视频的md5，也就是所谓的md5去重。
经过下面的2，4，5，6，7，8，9，10，11，12之后，基本上可以算是自己的视频了

## 程序使用

打开 config.toml 文件，修改配置，每一项都有一个开关，选择自己需要操作的选项。

运行

`./vm run`
